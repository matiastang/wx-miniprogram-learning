## number

数字键盘（无小数点），适用纯数字。

```html
<input type="digit"/>
```

## idcard

数字键盘（无小数点、有个 X 键），适用身份证。

```html
<input type="idcard"/>
```

## digit

数字键盘（有小数点），适用带小数的。

```html
<input type="digit"/>
```
